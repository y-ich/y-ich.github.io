Squeak 3.5

Squeakは、Smalltalk-80をベースにした急速推進中のプロジェクトです。依然としてSmalltalk-80とは適度に互換性があります。Squeakには、非常に統合された開発環境や、ネットワーキング、サウンド合成やサンプリング、音声合成、２次元／３次元グラフィックス、演算とデータ構造ライブラリが含まれます。。Squeakは、自身の仮想マシンのコアコードを生成するためのツールさえ含むので、あなたはあなた自身のVMを作ることができます。

Browser openBrower

[このウィンドウの中の青色の項目はアクティブテキストです。もし項目がURLを含むなら、クリックして内容を表示するためには、インターネット接続が必要です。また、ロードするのに若干時間がかかるかもしれません。]

すべてのソースコードは、含まれているだけでなく、思うままに変更可能です。また、完全にオープンでフリーです。Squeakシステムイメージはすべてのプラットフォームで全く同様に(bit-identically)動作します。VMは、入手可能なコンピュータとオペレーティングシステムのほぼすべてで利用可能です。Squeakプロジェクトの歴史は、

ftp://st.cs.uiuc.edu/Smalltalk/Squeak/docs/OOPSLA_Squeak.html

で知ることができます。

Squeakライセンスや役立つたいていの情報はSqueakホームページで見つけられます。

http://www.Squeak.org/

SqueakMap

最近利用可能になったエキサイティングな特徴は、Squeakのパッケージカタログシステムである、SqueakMapです。このリリースでは、Squeakコミュニティのメンバーによって、100以上のパッケージ／アプリケーションがSqueakMapに加えられました。SqueakMap Package Loaderを使うことで、これらのパッケージを簡単にブラウズしたり、直接Squeakにインストールしたりすることができます。SqueakMap Package Loaderは、"open..."/"Package Loader"メニューから（あるいは、ここをクリックすることで）起動できます。

SqueakMapに関する情報をもっと知りたい人は、

http://minnow.cc.gatech.edu/squeak/squeakmap.

Morphic

今回のSqueakのリリースではMorphicグラフィックアーキテクチャを採用しています。MVCプロジェクトの中ではMVCアーキテクチャも利用可能です。（ワールドメニューの'open...'オプションを見てください。）標準のシステムウィンドウのほとんどはどちらのフレームワークでも開くことができます。しかし、メディアサポートは主にMorphicで開発されています。更に、MorphicフレームワークはGenie（精霊）を含んでいます。GenieはペンだけでSqueakのすべてを利用することを可能にする、文字認識、ジェスチャ認識システムです。次の行をクリックしてみてください：

AGenieIntroduction.　

プロジェクト

プロジェクトはSqueakの中の個別の世界です。インターネット上のウェッブページに似ています。実際、アクティブなプロジェクトは、ちょうどウェッブページのようにインターネット経由で共有することができます。'Worlds of Squeak'ウィンドウの中に数多くのデモンストレーションプロジェクトを入れました。スクリーンの底辺にあるNavigatorタブの下の'FIND'ボタンを選択すると、ほかのプロジェクトを見つけることができるでしょう。

一度入ったプロジェクトから元に戻るには、ワールドメニューから'previous project'、もしくはナビゲータバーから'< PREV'を選択します。

カラーグラフィックス

SqueakのBitBltは可変色数など多くの性能強化が施されてきました。透かしをサポートしたペイントモードや32ビットカラーのためのアルファブレンドモードなどいくつかの機能が追加されました。ワンパスで拡大縮小、回転その他のビットマップ変形を行う”ワープドライブ”変形も組み込まれています。興味があるユーザーは、

Display restoreAfter: [WarpBlt test1]

や

Display restoreAfter: [WarpBlt test3]

を試してみたくなるでしょう。

サウンド

Squeakには、バックグラウンドで音や音楽をリアルタイム生成するためのベースクラスと簡単なプリミティブが含まれています。興味があるユーザーは、Worlds of Squeakの中の例や

AbstractSound stereoBachFugue play.

を試してみたくなるでしょう。

Balloon

今回、Squeakは、Balloonと名付けられた、全く新たなアウトラインベースのグラフィックスサブシステムを搭載しています。Balloonグラフィックスは拡大縮小や回転を考慮せず(is independent of)、単に描画するか、２度のアンチエーリアスで描画します。簡単なデモンストレーションを見るには、次をクリックしてください...

(FlashMorphReader on: (HTTPSocket

httpGet: 'http://www.squeak.org/Squeak3.4/areallycoolflash.swf'

accept: 'application/x-shockwave-flash'))

processFile startPlaying openInWorld.

（Herves Flores (Ｅメール - herve.flores@free.fr; ウェッブサイト - http://herve.flores.online.fr）のご好意による）この例は、SqueakがFlash3グラフィックスファイルフォーマットを（Balloonグラフィカルオブジェクトへ変換することで、まずまず完全に実装していることも示しています。すべてSqueakの中で実装されているので、プレーヤを止めて、morphic balloonオブジェクトを分解することができます。（注意：Balloon Flashプレーヤは、Flash3ファイルのみ再生します。――Flash4やFlash5は再生できません。）

ネットワーキング

Squeakはネットワーク接続をサポートしています。もしウェッブにアクセス可能なら、

HTTPSocket httpShowGif:

'http://squeak.org/Squeak2.0/midi/Squeakers.GIF'.

を実行してみたくなるでしょう。Socketクラスにはもっとたくさんの例が記述してあります。また、このリリースには完全なWikiWikiサーバが含まれています。WikiWikiに関する付属の情報を見てください。

SqueakのFileListはネットワークアクセスもサポートしています。どのサーバーが利用可能か[ ]ルートで見ることができます。また、'add server...'メニューコマンドで新たなサーバーを追加することができます。

最後に、既にお話したように、Squeakはネット経由でアクティブなプロジェクトをダウンロードできます...

Project fromUrl:

'http://209.143.91.36/super/SuperSwikiProj/FinalFrontier'.

開発サポート

Squeakには、完成度の高いプログラム開発開発システムが含まれています。このシステムの構成要素のほとんどは、ワールドメニューの'open...'オプションの中に見つけることができます。利用可能な機能(function)のほとんどは、色々なメニュー選択やワールドメニューの'help...' / 'command-key help'オプションで見渡すことができます。

Squeakの縮小化

もしこのイメージから"Worlds of Squeak"例をアンロードしたいなら、バックアップコピーを作り、以下の５行を実行して、ウィンドウを閉じて、そしてできるイメージを保存してください。

ProjectViewMorph allInstancesDo:

[:m | (m hasProperty: #deleteWorldOfSqueak) ifTrue:

[Project deletingProject: m project. m delete]].

Project rebuildAllProjects.

ScriptingSystem spaceReclaimed

これによりかなり縮小化することができます。Squeakの以前のリリースでは、特別に用意された、"MajorShrinkFor3.0.cs"のような名前のファイルインが一緒に配布されていました。それは、同封の説明書に書かれているように使うと、1MB以下サイズのイメージを生成します。Squeak 3.2ではこのようなmajorShrinkファイルインは用意されていませんが、出発点として以前のバージョンのmajorShrinkを使うことができます。あるいは、Squeak swiki上で縮小化に関する最新情報を探したほうがよいかもしれません。よい出発点として下記のページを参照してください：

http://minnow.cc.gatech.edu/squeak/majorshrink
